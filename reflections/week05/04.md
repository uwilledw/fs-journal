## Questions

1. What is a ***virtual property***?

A virtual property is a object that is added on to a model but isn't in that model.

2. When might you use a virtual property? 

You would use a virtual property when you want to add some data to an existing model so you can access it.

3. How do you search by a virtual property's value?

You use a setter and alias out the the value.
